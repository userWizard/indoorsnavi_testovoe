# Техническое задание

## Описание проекта

Разработка простейшего сервиса для управления базой данных котов с веб-интерфейсом для заводчиков, использующего Django, Django Rest Framework и Angular 8+.

## Требования к функционалу

1. **Регистрация пользователей**
   - Пользователь может зарегистрироваться как заводчик кота.

2. **Управление котами**
   - Создание кота.
   - Удаление кота.
   - Редактирование параметров кота (например, имя, возраст, порода, волосатость).
   - Пользователь может управлять только своими котами.

3. **Раздел "Сообщения"**
   - Отправка и получение сообщений другим заводчикам через вебсокеты.

4. **Интерфейс**
   - Использование Angular Material или любого другого фреймворка для создания интерфейса.

5. **Деплоймент**
   - Развертывание приложения на сервере Ubuntu с доступом к интернету (например, Heroku).

6. **Docker**
   - Упаковка приложения в Docker-контейнер.

## Технологии
- **Backend:** Django, Django Rest Framework
- **Frontend:** Angular 8+
- **Дополнительно:** Websockets для сообщений

## Docker
- Создание Dockerfile для сборки проекта.
- Конфигурация Docker Compose для локальной разработки и тестирования.

## Запуск приложения
1. Клонировать репозиторий проекта.
2. Собрать контейнеры с помощью Docker.
3. Развернуть приложение на сервере.

---

> **Примечание:** Использование Angular Material не будет оцениваться, предоставляется на выбор для удобства разработки интерфейса.
